# 🤖 Optimisation AI Agent

Bienvenue dans l'Optimisation AI Agent, une plateforme complète conçue pour orchestrer des LLMs (Locale ou Cloud) afin de réaliser des actions système complexes : gestion de fichiers, exécution de commandes OS et web scraping.

Ce projet est structuré en trois couches principales, allant de l'interface utilisateur graphique à la couche d'exécution sécurisée.
